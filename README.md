# リプレイファイルダウンロード

## 概要
リアルタイム地震ビューアーで使用できるリプレイファイルをダウンロードできるプロジェクトです。


## 使用方法
本ページにアクセスし、地震の発生時刻または検知時刻順に並べられているので、対象の地震を選択してダウンロードします。  
ここからの方法は既にリアルタイム地震ビューアー改造版で説明されているので省略します。

## データに関して
強震観測網のデータは再配布禁止なので、基本的に過去の地震は配信されません。  
また、全ての地震をファイル化するわけではありません。規模・震度によってファイル化するかを独断で決めています。  
なお、こちらの都合により規模・震度が大きい地震でもファイル化されない場合があります。
