# リプレイファイルダウンロード

## 概要
リアルタイム地震ビューアーで使用できるリプレイファイルをダウンロードできるプロジェクトです。


## 使用方法
本ページにアクセスし、地震の発生時刻または検知時刻順に並べられているので、対象の地震を選択してダウンロードします。  
ここからの方法は既にリアルタイム地震ビューアー改造版で説明されているので省略します。

## データに関して
強震観測網のデータは再配布禁止なので、基本的に過去の地震は配信されません。  
また、全ての地震をファイル化するわけではありません。規模・震度によってファイル化するかを独断で決めています。  
なお、こちらの都合により規模・震度が大きい地震でもファイル化されない場合があります。

## 改造版の旧利用条件について
このリプレイファイルダウンロード内にはリアルタイム地震ビューアー改造版で配布されていたリプレイファイル、またはそれを改変したものが含まれます。  

### 利用条件
リアルタイム地震ビューアー改造版には、リプレイファイルの利用条件なるものがありました。    
この利用条件では2025/12/10に改造版上でのリプレイファイルの配布が一時停止されるまで、**他人の作品独自のフォーマットで、創造的なものだったファイル**であったにもかかわらず、「再配布禁止」等とCreative Commons　Attribution 2.0で禁止されている追加の制限を課していました。  
この利用条件自体が違反なため、この利用条件は無効です。そのためこのデータは再配布可能です。  
なお利用条件がCCBYライセンス2.0に違反している根拠は以下の通りです。

### 根拠
リプレイファイルは、ソフトウェアによって生成されるデータ、あるいは作品製作者が一部、または全て創作した内容を含むデータで構成されている。この点から、現在のように各ファイルごとにライセンス表記がなく、独自の利用条件を課している状態は、CC BY 2.0 ライセンスに抵触する可能性が高い。
CC BY 2.0 は共有・改変を広く許諾する一方で、追加の制限（例：固有の利用規約や制約）を付与することを禁じているためである。
また、リプレイファイルが製作者の創造的著作物に該当すると考えられる根拠は以下の通り。
1. データ形式が作品制作者独自のフォーマットであること
2. 公的機関等の情報ではなく、作品製作者が独自に生成した予想震度などのデータが一部に含まれていること
以上から、現在のようなリプレイファイル配布形式は、CC BYライセンス2.0の違反になる可能性が高い。
